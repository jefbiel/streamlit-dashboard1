# streamlit-dashboard1

Projeto Streamlit para apresentacao pessoal e analise de dados.

## Rodar localmente

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m streamlit run app.py
```

## Fluxo rapido de commit e push

```bash
chmod +x git-flow.sh
./git-flow.sh "mensagem do commit"
```

## Deploy no Streamlit Community Cloud

1. Acesse https://share.streamlit.io/
2. Clique em New app.
3. Selecione o repositorio `jefbiel/streamlit-dashboard1`.
4. Em Main file path, use `app.py`.
5. Clique em Deploy.
